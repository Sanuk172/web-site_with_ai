# AI Ассистент

Этот раздел описывает поведение встроенного ассистента и его роль на портале.

## Назначение

Ассистент помогает пользователю быстро находить ответы по материалам базы знаний
и сокращает время поиска нужной страницы.

## Ожидаемый формат ответа

- короткий и точный ответ на вопрос;
- ссылки на страницы-источники внутри базы;
- отсутствие домыслов, если данных недостаточно.

## Техническая схема (план)

1. Индексация Markdown-страниц.
2. Поиск релевантных фрагментов по эмбеддингам.
3. Генерация ответа только по найденному контенту.
4. Возврат ответа вместе со списком источников.

## Текущее состояние

Интерфейсная часть подготовлена. Предметный корпус по ИБ и данные индекса будут
подключены после наполнения основной базы знаний.
