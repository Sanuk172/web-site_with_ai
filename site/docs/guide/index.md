# Обзор структуры

Раздел описывает архитектуру контентной части и принципы наполнения сайта.
Здесь собраны базовые страницы, от которых удобно стартовать перед загрузкой
полноценной базы знаний по информационной безопасности.

## Назначение разделов

- `section-1` - блок для теоретических материалов и вводных тем;
- `section-2` - блок для прикладных сценариев, чеклистов и практики.

## Рекомендации по наполнению

1. Одна страница = одна понятная тема.
2. Начинайте статью с краткой цели и целевой аудитории.
3. Используйте одинаковую структуру заголовков во всех материалах.
4. Добавляйте ссылки на связанные статьи внутри базы.

## Текущее состояние

Каркас готов для масштабирования. Контент по ИБ в эти страницы пока намеренно
не добавлен, чтобы команда могла заполнить его централизованно и согласованно.
