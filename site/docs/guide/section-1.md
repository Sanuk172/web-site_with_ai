# Раздел 1

Раздел предназначен для базовых и вводных материалов.

## Что планируется размещать

- определения ключевых терминов;
- обзор стандартов и ролей;
- базовые модели угроз;
- стартовые рекомендации для новичков.

## Формат будущих статей

Каждая статья должна содержать цель, контекст применения, основной материал и
короткий вывод. Такой формат позволит ассистенту точнее извлекать факты и
ссылаться на корректные источники.

## Примечание

Содержательная база по ИБ в этот файл пока не добавляется, страница служит
подготовленной оболочкой для дальнейшего наполнения.
