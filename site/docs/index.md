## О проекте

Этот сайт является витриной учебного проекта "База знаний с AI-ассистентом".
Сейчас в нём настроены навигация, внешний вид, структура разделов и страница для
интеллектуального помощника. Предметный контент по информационной безопасности
будет добавляться постепенно отдельными материалами.

## Что уже подготовлено

- единый стиль интерфейса и брендированный внешний вид;
- базовая структура разделов для будущих статей;
- заготовка под AI-ассистента с поддержкой ответов на основе контента;
- готовность к workflow "контент как код" через Git.

## Как использовать каркас

1. Перейдите в раздел `Разделы` и выберите нужную ветку.
2. Откройте страницу-шаблон из справки.
3. Добавляйте новые статьи по теме ИБ в формате Markdown.
4. После публикации ассистент сможет использовать эти данные в ответах.
