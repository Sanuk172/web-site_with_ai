# Шаблон статьи

## Заголовок

Кратко сформулируйте тему и ценность материала для читателя.

## Цель материала

Опишите, какой вопрос решает статья и кому она адресована.

## Основная часть

Разбейте материал на 2-4 логических блока с подзаголовками.

## Краткий итог

Сформулируйте 2-3 ключевых вывода.

## Источники и связанные страницы

Добавьте внутренние ссылки на другие статьи базы и, при необходимости,
внешние нормативные документы.
